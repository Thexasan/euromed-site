# Даво Сифат — Медицинский центр в Душанбе

Официальный сайт клиники Даво Сифат — современного медицинского центра в городе Душанбе, Таджикистан.

**Сайт:** [euromed-site.vercel.app](https://euromed-site.vercel.app)

---

## О проекте

Статический сайт-визитка медицинской клиники Даво Сифат, включающий информацию об услугах, специалистах, новостях и форму онлайн-записи на приём.

### Разделы сайта

- **Hero** — главный баннер с краткой информацией о клинике
- **Статистика** — ключевые цифры (15+ лет опыта, 40+ специалистов, 10 000+ пациентов в год)
- **Услуги** — кардиология, педиатрия, неврология, стоматология, гинекология, УЗИ-диагностика, лаборатория, терапия
- **Специалисты** — карточки врачей (Умар Азизов, Нилуфар Рашидова, Бахром Турсунов, Мадина Юсупова, Сухроб Мирзаев, Зарина Холматова)
- **Онлайн-запись** — форма записи на приём с выбором специалиста и даты
- **Новости** — последние новости и медицинские рекомендации
- **Футер** — контакты, навигация, юридическая информация

## Деплой

Проект автоматически деплоится на **Vercel** при каждом push в ветку `main`:

1. Внесите изменения в файлы
2. Сделайте commit и push в `main`
3. Vercel автоматически задеплоит обновление

## Адаптивность

Сайт адаптирован под все устройства:

- **Desktop** (> 992px) — полная навигация, 4-колоночные сетки
- **Tablet** (601–992px) — бургер-меню, 2-колоночные сетки
- **Mobile** (< 600px) — мобильный drawer, 1-колоночная вёрстка

Реализована поддержка Safari (iOS): `-webkit-sticky`, `-webkit-appearance`, `-webkit-backdrop-filter`.
